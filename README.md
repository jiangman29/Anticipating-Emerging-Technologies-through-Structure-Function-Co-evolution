Supplementary data and code for structure–function extraction, BERTopic topic identification, and technology cluster mapping in digital medical technologies
This repository provides the supplementary materials for the study on structure–function co-evolution and technological emergence in digital medical technologies. It contains three appendices covering the Prompt-based structure–function extraction method, the extracted structure–function data, and the BERTopic-based topic modeling and clustering results.

Contents

Appendix 1. Prompt code for structure–function semantic extraction
This appendix contains the Prompt templates, output specifications, and implementation code for structure–function semantic extraction. The extraction framework includes four fields: Structure, Function, S–S, and S–F.

Appendix 2. Structure–function extraction results for digital medical technologies
This appendix provides the extracted structure–function data for digital medical technology patents. For each patent, the extracted information includes S, F, SS, and SF.

Appendix 3. BERTopic results for digital medical technologies
This appendix contains the BERTopic-based topic modeling results for digital medical technologies. The analysis identified 66 main technology topics and 99 subtopics, forming a fine-grained topic system with 165 technology topics in total. The appendix includes topic IDs, topic names, representative keywords, and the assignment of all topics into 12 technology clusters, with the clustering results provided in a separate column.
